## 📋 Overview

Automated Telegram bot that monitors eBay Kleinanzeigen listings and sends instant notifications about new ads. Perfect for immigrants in Germany, deal hunters, and anyone looking for second-hand bargains.

**Never miss a great deal again!** Get real-time alerts when new listings match your search criteria, with automatic translation and price tracking.

## ✨ Features

### Core Features
- 🔔 **Real-time notifications** - Get alerts within minutes of new listings
- 🌍 **Multi-language support** - Interface in German, Russian, Ukrainian, English
- 🔄 **Auto-translation** - Translate German listings to your language using Claude AI
- 📊 **Price tracking** - Monitor price changes over time
- ⏰ **Custom intervals** - Set how often to check (1min - 1hour)
- 🎯 **Smart URL tracking** - Just paste eBay Kleinanzeigen search URL with your filters

### Analytics Features
- 📈 **Price history graphs** - See how prices changed
- 💡 **Bargain meter** - "Item listed for 30+ days, ask for 20% discount!"
- 💰 **Deal scoring** (coming soon) - AI rates deal quality 0-100
- 💵 **Fair market value** (coming soon) - See if price is fair
- ⚠️ **Scam detection** (coming soon) - Flags suspicious listings

### User Experience
- 💾 **Save for later** - Bookmark interesting listings
- 🚫 **Hide forever** - Never see certain ads again
- 🔍 **Keyword highlights** - Important words highlighted in notifications
- 📊 **/stats** - Track your activity and notifications
- 💬 **Support button** - Request features directly from the bot

## 💡 Use Cases

- 🏠 **Real Estate Hunters** - Get notified of new apartments instantly in Berlin, Munich, Hamburg
- 🚗 **Car Buyers** - Track used car prices (Mercedes, BMW, VW) and find deals
- 💻 **Tech Enthusiasts** - Monitor electronics (MacBooks, iPhones, gaming PCs, etc.)
- 🛋️ **Furniture Shoppers** - Find cheap IKEA furniture for students and expats
- 👕 **Fashion Deals** - Track designer clothes at discount prices
